AutoPlus has a variety of slash commands that you can use to interact with the bot. Here is a list of all these commands and their descriptions.

## Escrow Related commands [#escrow-related]

These commands are related to the escrow system of AutoPlus.

<ul className="list-none list-inside">
  <li>`/trade <user> <cryptocurrency>`: Start a trade with a user. This can be very useful to open a trade quickly.</li>
  <li>`/history`: View your trade history.</li>
  <li>`/add user`: Add a user to the trade. Rest assures, the user won't be able to release or return the funds.</li>
  <li>
    `/settings`: Change your settings.
    <ul className="list-none list-inside ml-6 mt-2">
      <li>Toggle anonymous mode: This will hide your username from public logs</li>
      <li>Toggle password: This will ask you a password each time you will want to release/return funds. This can be very useful in case your Discord account gets hijacked</li>
      <li>Set default addresses: This will set your default addresses for each cryptocurrency so that you don't have to type them each time you want to claim your funds.</li>
      <li>Recovery token: This will generate a recovery token that you will NEED to recover your AutoPlus account in case you lose access to your current Discord account.</li>
    </ul>
    <p>The recovery token is the only proof that you are the owner of the account. If you lose it, you will lose access to your account and we won't be able to help you. Please keep it safe.</p>
  </li>
</ul>

## Points Related commands [#points-related]

These commands are related to the points system of AutoPlus.

<ul className="list-none list-inside">
  <li>`/balance`: View your points balance and your referral balance if you have one.</li>
  <li>`/tip`: Tip a user with points.</li>
  <li>`/blackjack`: Play a game of blackjack with the bot.</li>
  <li>`/coinflip`: Play a game of coinflip with a user.</li>
  <li>`/mines`: Find multiple gems in a row to win points.</li>
  <li>`/plinko`: Drop a ball and win points.</li>
</ul>

## Miscellaneous commands [#miscellaneous]

These commands are miscellaneous commands that don't fit in the other categories.

<ul className="list-none list-inside">
  <li>`/leaderboard`: Get the leaderboard of the bot. You can see the leaderboard based on the number of trades, the amount sent, amount received and the points balance.</li>
  <li>`/stats user`: Get the stats of a user. You can see the number of trades, the amount sent, amount received and the points balance of a user.</li>
  <li>`/stats bot`: Get the stats of the bot. You can see the total number of trades, volume, canceled trades, canceled volume, the total number of users and the points supply.</li>
</ul>
